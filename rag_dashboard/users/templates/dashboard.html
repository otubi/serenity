Set up your API key

For initial testing, you can hardcode an API key, but this should only be temporary since it is not secure. The rest of this section explains how to set your API key locally for different operating systems.

Linux/macOS - Bash

Bash is common for Linux and macOS terminals. To check for a configuration file, run:

~/.bashrc

If it doesn't exist, create and open it:

touch ~/.bashrc
open ~/.bashrc

Add your API key:

export SERENITY_API_KEY=<YOUR_API_KEY_HERE>

Apply the changes:

source ~/.bashrc

Send your first Serenity API request using FastAPI

You can use FastAPI to create a simple API client like this:

from fastapi import FastAPI, Header, HTTPException
import requests

app = FastAPI()

API_KEY = "YOUR_API_KEY_HERE"

@app.get("/get-models")
def get_models(api_key: str = Header(...)):
if api_key != API_KEY:
raise HTTPException(status_code=403, detail="Invalid API Key")

response = requests.get(
"https://api.serenityai.com/v1/models",
headers={"Authorization": f"Bearer {api_key}"}
)
return response.json()

Run your FastAPI app using:

uvicorn main:app --reload
